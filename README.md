# Android Lab 3 - SimpleWordle

Submitted by: Eugene Shanga

National_Parks is an application that gets all the national parks and some information from them as described by the national parks service. Visit nps.gov to learn more!
Time spent: 3 hours spent in total

## Required Features

The following **required** functionality is completed:

- [ ] **Live data is loaded from the National Parks API**
- [ ] **Parks are displayed using a RecyclerView**
- [ ] **Park images are downloaded and displayed using Glide**
